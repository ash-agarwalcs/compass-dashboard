One-time setup required

To enable Google Sign-In, you need a Google Cloud OAuth Client ID. This is a one-time setup:

  1. Go to Google Cloud Console → Credentials
  2. Create a project (or pick an existing one)
  3. Enable the Google Sheets API under "APIs & Services"
  4. Create an OAuth 2.0 Client ID (type: Web application)
  5. Add your hosting domain to Authorized JavaScript origins (e.g. http://localhost:8000 for local dev)
  6. Copy the Client ID and paste it in this file at the top of the <script> section:
    const CLIENT_ID = 'your-id.apps.googleusercontent.com';
